Algorithm

Root Mean Squared Error

Description

RMSE is the square root of the average of squared differences between prediction and actual observation. It represents the sample standard deviation of the differences between predicted values and observed values.

$$ \text{RMSE} = \sqrt{\frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2} $$
