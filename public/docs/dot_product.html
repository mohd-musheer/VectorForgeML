Algorithm

Dot Product

Description

Computes the dot product of two vectors using BLAS Level 1 operations. Optimized for speed using assembly-level instructions where available via the BLAS backend.

$$ \mathbf{a} \cdot \mathbf{b} = \sum a_i b_i $$

Implementation Details

Implemented in `functions.cpp`.

F77_CALL(ddot)(&n, a.begin(), &inc, b.begin(), &inc)

Use Cases

General purpose.
