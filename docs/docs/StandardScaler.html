Algorithm

Standard Scaler

Description

Standardizes features by removing the mean and scaling to unit variance.

Parameters

NameTypeDescription
xMatrixInput feature matrix.
yVectorTarget vector.
......Additional parameters.

Usage Example

# Load library
library(VectorForgeML)

# Prepare data
X <- matrix(rnorm(100), ncol=5)
y <- rnorm(20)

# Initialize and train
model <- StandardScaler()
model$fit(X, y)

# Predict
preds <- model$predict(X)

Implementation Details

Implemented in `scalers.R`. Uses `cpp_scale_fit_transform` for speed if available, otherwise R `scale`.

fit=function(X){
  X <- as.matrix(X)
  if (exists("cpp_scale_fit_transform", mode="function")) {
    out <- cpp_scale_fit_transform(X)
    mean <<- out$mean
    sd <<- out$sd
  } else {
    mean <<- colMeans(X)
    sd <<- apply(X,2,sd)
    sd[sd==0] <<- 1
  }
}
