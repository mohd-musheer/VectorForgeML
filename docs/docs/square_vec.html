Algorithm

Square Vector

Description

Computes the element-wise square of a vector. This is a common operation in variance calculations.

$$ y_i = x_i^2 $$

Parameters

NameTypeDescription
xMatrixInput feature matrix.
yVectorTarget vector.
......Additional parameters.

Usage Example

# Load library
library(VectorForgeML)

# Prepare data
X <- matrix(rnorm(100), ncol=5)
y <- rnorm(20)

# Initialize and train
model <- square_vec()
model$fit(X, y)

# Predict
preds <- model$predict(X)
