Algorithm

One Hot Encoder

Description

Encodes categorical features as a one-hot numeric array.

Parameters

NameTypeDescription
xMatrixInput feature matrix.
yVectorTarget vector.
......Additional parameters.

Usage Example

# Load library
library(VectorForgeML)

# Prepare data
X <- matrix(rnorm(100), ncol=5)
y <- rnorm(20)

# Initialize and train
model <- OneHotEncoder()
model$fit(X, y)

# Predict
preds <- model$predict(X)
